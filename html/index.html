Introduction

This program is an Arduino library for NS73M low power FM transmitter.
NS73M is a low power FM transmitter controlable by I2C or SPI bus (only software SPI is used in this library).
It can be used like any chinese FM car transmitter but with full control of transmission specifications.
This include power management, input volume level, emphasis, target frequency, and much other !

Please report bug to <skywodd at gmail.com>

Note:

How to wire the breakout board

Take a look at the image below to know how to wire your breakout board with your arduino.


Breakout NS73M (3v3) -> Arduino UNO (5v)
CK -> 10K resistor -> Digital 12 (by default)
DA -> 10K resistor -> Digital 11 (by default)
LA -> 10K resistor -> Digital 10 (by default)
IIC -> GND Power Rail
VCC -> 3V3 Power Rail
GND -> GND Power Rail

Where to buy this breakout board ?

This library is tested with the following breakout boards :

  • FM Radio Module Breakout Board (Sparkfun) : http://www.sparkfun.com/products/8482

Arduino versions compatibility

This library use only basic arduino functions like pinMode() and digitalWrite().
By this way this library is virtualy compatible with any version of Arduino IDE.

Remarks:
This library SHOULD work on boards derivated from arduino project like chipkit or maple.
